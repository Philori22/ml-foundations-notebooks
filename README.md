# ML Foundations Notebooks

A small collection of notebook exercises for refreshing and strengthening machine learning fundamentals through implementation.

This repo is designed for people who want to build a stronger first-principles understanding of core ML concepts by writing code, checking assumptions, and reasoning about results.

## What this repo covers

The notebooks focus on practical foundations such as:

- Python and NumPy fluency
- dataset inspection and summary statistics
- mean, variance, and standardisation
- train/test splitting
- simple helper functions used in ML workflows
- shape-checking and debugging habits
- core modelling intuition

Over time, this repo may expand to include:

- linear regression
- logistic regression
- PyTorch training loops
- regularisation and validation
- representation learning basics
- sequence modelling and dynamics prediction

## How to use this repo

Each notebook is intended to be completed actively rather than read passively.

Recommended approach:

1. Open a notebook and read the task carefully.
2. Try to explain the goal in your own words before coding.
3. Complete the `TODO` sections yourself.
4. Test small pieces as you go.
5. Print shapes and intermediate outputs often.
6. Reflect on both the implementation and the underlying concept.

The aim is not just to get the notebook running, but to build fluency in:
- implementation
- debugging
- reasoning
- communication

## Intended audience

This repo may be useful for:

- students learning ML fundamentals
- engineers refreshing core concepts
- researchers who want cleaner implementation habits
- anyone practising applied ML from first principles
